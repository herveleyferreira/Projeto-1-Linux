# __Projeto 1 Linux - DEVSECOPS Compass UOL__
### Atividade prática realizada no programa de bolsas Compass UOL na trilha DevSecOps.
## Objetivos:

- Criar um subsistema Linux no Windows usando WSL com Unbutu 20.04 ou superior.
- Instalar e configurar o servidor Nginx para estar online e acessível.
- Criar um script que valide o status do servidor Nginx: _o script deve conter - Data + Hora + nome do serviço + Status + mensagem personalizada de **online** ou **offline**_
- Configurar a execução automática do script a cada 5 minutos.
- Fazer o versionamento e a documentação do projeto.
